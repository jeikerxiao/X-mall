# X-mall

x-mall 是一个商城的学习项目，目前这个项目只有后台接口。

# 项目技术点

* Spring Boot
* MyBatis
* MySQL
* Swagger2 

# 项目说明

项目的后台接口分前台和后台两套接口组成。

* backend (后台接口)
* frontend (前台接口)

# 接口地址

http://localhost:8080/swagger-ui.html

项目使用 swagger2 列出所有接口列表，可以直接在上面的网页面进行调试接口。

# 运行项目 

1. 导入数据库

2. 修改项目中数据库连接配置 

3. 运行项目

4. 打开 http://localhost:8080/swagger-ui.html 调试项目
